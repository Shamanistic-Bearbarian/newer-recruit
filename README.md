# newer-recruit

A web-based army list builder for **Warhammer 40,000 (11th edition)** — inspired by [New Recruit](https://www.newrecruit.eu/).

> ⚠️ 11th edition is not yet released. Until official data exists, this app ships with clearly-marked **placeholder** data for development. No invented stats represent real game rules.

## Status
Early scaffolding. See issues / commits for progress.
